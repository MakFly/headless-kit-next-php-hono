## Overview

The SaaS module provides multi-tenant organization management: plans and subscriptions, invoices, team member management, usage tracking, and organization settings.

## API Endpoints

### Organizations

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/orgs` | Yes | List user's organizations |
| `POST` | `/api/v1/saas/orgs` | Yes | Create organization |
| `GET` | `/api/v1/saas/orgs/{id}` | Yes | Show organization details |

### Plans & Subscriptions

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/plans` | No | List available plans |
| `GET` | `/api/v1/saas/orgs/{orgId}/subscription` | Yes | Get current subscription |
| `POST` | `/api/v1/saas/orgs/{orgId}/subscribe` | Yes | Subscribe to a plan |
| `POST` | `/api/v1/saas/orgs/{orgId}/cancel` | Yes | Cancel subscription |

### Billing

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/orgs/{orgId}/invoices` | Yes | List invoices |

### Team

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/orgs/{orgId}/team` | Yes | List team members |
| `POST` | `/api/v1/saas/orgs/{orgId}/team/invite` | Yes | Invite member |
| `PATCH` | `/api/v1/saas/orgs/{orgId}/team/{id}/role` | Yes | Change member role |
| `DELETE` | `/api/v1/saas/orgs/{orgId}/team/{id}` | Yes | Remove member |

### Usage & Settings

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/orgs/{orgId}/usage` | Yes | Get usage stats |
| `GET` | `/api/v1/saas/orgs/{orgId}/settings` | Yes | Get org settings |
| `PUT` | `/api/v1/saas/orgs/{orgId}/settings` | Yes | Update settings |

### Dashboard

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/saas/orgs/{orgId}/dashboard` | Yes | Org dashboard stats |
