Four skills created in `apps/api-laravel/.claude/skills/`:

- `lara-action` — Create an invokable Action class for a single endpoint
- `lara-feature` — Scaffold a full feature slice (Actions + routes.php + optional Requests/Services/Formatters)
- `lara-test` — Write Feature or Unit test classes
- `lara-migration` — Create and run database migrations

**Why:** Provides quick scaffolding templates so future agents don't need to rediscover conventions each time.

**How to apply:** Reference these skills when generating new Laravel code in `apps/api-laravel/`.
