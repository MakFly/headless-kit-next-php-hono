## Overview

The Support module provides a help desk system: conversation threads, real-time messaging, canned responses for agents, agent queue and assignment, and customer satisfaction ratings.

## API Endpoints

### Conversations (Customer)

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/support/conversations` | Yes | List user's conversations |
| `POST` | `/api/v1/support/conversations` | Yes | Create new conversation |
| `GET` | `/api/v1/support/conversations/{id}` | Yes | Show conversation with messages |
| `POST` | `/api/v1/support/conversations/{id}/messages` | Yes | Send message |
| `POST` | `/api/v1/support/conversations/{id}/rate` | Yes | Rate conversation |

### Agent Queue

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/support/agent/queue` | Agent | Unassigned conversations |
| `GET` | `/api/v1/support/agent/assigned` | Agent | Agent's assigned conversations |
| `POST` | `/api/v1/support/agent/conversations/{id}/assign` | Agent | Assign to self |
| `PATCH` | `/api/v1/support/agent/conversations/{id}/status` | Agent | Update status (open/resolved/closed) |

### Canned Responses

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/support/canned-responses` | Agent | List canned responses |
| `POST` | `/api/v1/support/canned-responses` | Agent | Create canned response |
| `PUT` | `/api/v1/support/canned-responses/{id}` | Agent | Update canned response |
| `DELETE` | `/api/v1/support/canned-responses/{id}` | Agent | Delete canned response |

### Analytics

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/support/ratings/stats` | Agent | Satisfaction rating stats |
