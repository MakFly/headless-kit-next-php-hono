## What is Headless Kit?

Headless Kit is an opinionated starter kit that pairs a **BFF** (Backend-for-Frontend) with your choice of backend: **Laravel**, **Symfony**, or **Hono**. Choose between **Next.js** (recommended) or **TanStack Start** as your frontend -- both implement the same secure BFF proxy pattern. You get production-ready domain modules (Auth, Shop, SaaS, Support), role-based access control, and a secure proxy layer from day one.

## Quick Start

```bash
bunx create-headless-app my-app
cd my-app
bun install
bun run dev
```

The CLI scaffolds the full monorepo with your choice of **preset** (E-commerce, SaaS, Admin, Landing), **frontend** (Next.js or TanStack Start), and **backend** (Laravel, Symfony, Hono).
