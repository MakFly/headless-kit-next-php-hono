## Installation

No global install needed — run directly with `bunx`:

```bash
bunx create-headless-app my-app
```

## Interactive Prompts

The CLI walks you through 6 prompts to configure your project:

<Steps>

1. **Project name**

   ```
   ? Project name: my-app
   ```

   Lowercase letters, numbers, and hyphens only.

2. **Backend**

   ```
   ? Backend
     ● Laravel   — PHP 8.3 + BetterAuth + RBAC
     ○ Symfony   — PHP 8.3 + BetterAuth + Paseto V4
     ○ Hono      — Bun + Drizzle, lightweight
   ```

3. **Frontend**

   ```
   ? Frontend
     ● Next.js       — App Router + shadcn/ui (recommended)
     ○ TanStack Start — Vite + TanStack Router + Zustand
   ```

4. **UI Preset**

   ```
   ? UI Preset
     ● E-commerce  — Store + cart + admin
     ○ SaaS        — Landing + admin + billing
     ○ Admin       — Dashboard + RBAC
     ○ Landing     — Marketing + auth
     ○ None        — Minimal BFF only
   ```

   See [Presets](/cli/presets/) for details on what each preset includes.

5. **Additional modules** (multiselect)

   ```
   ? Modules (space to select)
     ◻ Multi-tenant organizations
     ◻ Billing (requires multi-tenant)
     ◻ Team management (requires multi-tenant)
     ◻ Org-level RBAC (requires multi-tenant + team)
     ◻ Support chat
     ◻ AI Assistant
   ```

   Some modules have dependencies — the CLI enforces them automatically.

6. **Database**

   ```
   ? Database
     ● SQLite      — Zero config, great for development
     ○ PostgreSQL  — Production-ready
   ```

</Steps>

## What Gets Scaffolded

After answering the prompts, the CLI:

1. Copies the base monorepo structure (turbo.json, root package.json)
2. Copies the selected frontend template to `apps/web/`
3. Copies the selected backend template to `apps/api/`
4. Applies the UI preset overlay (pages, components, styles)
5. Applies selected module overlays (frontend + backend)
6. Generates `.env` files with secure random secrets
7. Generates `docker-compose.yml` for the selected backend

## Post-Scaffold Steps

```bash
cd my-app
bun install

# If using Laravel or Symfony:
cd apps/api-laravel    # or apps/api-sf
composer install
php artisan migrate --seed   # or php bin/console doctrine:migrations:migrate

# Start everything:
cd ../..
bun run dev
```

<Aside type="tip">
  The CLI generates secure random values for `BFF_SECRET`, `JWT_SECRET`, `BETTER_AUTH_SECRET`, and `APP_KEY` automatically. You don't need to set them manually for development.
</Aside>

## Project Structure

The scaffolded project follows the standard Headless Kit monorepo structure:

```
my-app/
├── apps/
│   ├── web/              # Frontend (Next.js or TanStack Start)
│   └── api-{backend}/    # Backend (Laravel, Symfony, or Hono)
├── package.json          # Root workspace config
├── turbo.json            # Turborepo pipeline
├── docker-compose.yml    # Docker setup for selected backend
└── bun.lock
```
