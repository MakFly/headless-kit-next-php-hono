## Overview

The Shop module provides a complete e-commerce backend: product catalog, categories, shopping cart, order management, and product reviews. It is implemented across all 3 backends using Vertical Slice Architecture.

## API Endpoints

### Products

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/shop/products` | No | List products (paginated, filterable) |
| `GET` | `/api/v1/shop/products/{id}` | No | Show product with reviews |

### Categories

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/shop/categories` | No | List all categories |
| `GET` | `/api/v1/shop/categories/{id}` | No | Show category with products |

### Cart

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `GET` | `/api/v1/cart` | Yes | Get current user's cart |
| `POST` | `/api/v1/cart/items` | Yes | Add item to cart |
| `PATCH` | `/api/v1/cart/items/{id}` | Yes | Update item quantity |
| `DELETE` | `/api/v1/cart/items/{id}` | Yes | Remove item from cart |

### Orders

| Method | Endpoint | Auth | Description |
| ------ | -------- | ---- | ----------- |
| `POST` | `/api/v1/orders` | Yes | Create order from cart |
| `GET` | `/api/v1/orders` | Yes | List user's orders |
| `GET` | `/api/v1/orders/{id}` | Yes | Show order details |

| -------- | ------------ | --------- |
| Next.js | `src/lib/actions/shop/` | Server actions using BFF client |
| TanStack Start | `src/lib/services/shop-service.ts` | Client-side fetch via BFF proxy |
